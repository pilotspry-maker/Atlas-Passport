What it does:

- Drops `Public join waitlist` (`{public}` INSERT, `WITH CHECK (true)`).
- Replaces with `anon_join_waitlist`:
  - `TO anon` only
  - `WITH CHECK` requires non-empty, regex-matching `email` (length 3‚Äì320)

- `REVOKE UPDATE (invited, position_tier) FROM anon, authenticated;` (defense-in-depth)
- `REVOKE INSERT FROM authenticated;` (waitlist is signed-out only)
- Idempotent. Preflight `RAISE EXCEPTION` if migration 024 has not been applied first, so out-of-order application fails loudly.
